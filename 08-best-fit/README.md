# Experiment 8: Best Fit

## Aim
To implement Best Fit memory allocation algorithm.

## Algorithm
1. Read memory block sizes.
2. Read process sizes.
3. Allocate each process to the smallest suitable block.
4. Display allocation.

## Input
Number of blocks: 5  
Number of processes: 4

Block Sizes:
100 500 200 300 600

Process Sizes:
212 417 112 426

## Sample Output
Process No   Process Size   Block No

1            212            4  
2            417            2  
3            112            3  
4            426            5
